About

CMFP states for Connected Markets Financial Platform. CMFP is a multi-asset event-driven asynchronous core, modern customizable ui and integration framework with pluggable adapters for liquidity, market and reference data sources. CMFP provides IDM, Account Management, OMS, ORS, Strategy and other services.
